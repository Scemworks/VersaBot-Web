Welcome to VersaBot

Your friendly Discord bot built with Python!

Commands

/ping

Ping and get latency of the bot.

/dice

Rolls a dice and shows the result.

/flip

Flips a coin and shows Heads or Tails.

/qr

Generates a QR Code from a given link/text.

/fortune

Get a random fortune.

/tarot

Draw a tarot card with an emoji!

/help

Get help with the bot and see available commands.

How to Run the Bot

Step 1

Clone the repository from GitHub:
git clone https://github.com/Scemworks/VersaBot.git

Step 2

Change to the directory: cd VersaBot

Step 3

Install the required packages: pip install -r requirements.txt

Step 4

Set up your Discord bot and get the token.

Step 5

Create a .env file with your token: TOKEN=your_token_here

Step 6

Run the bot with: python your_bot_file.py
