Quick Setup Guide

Get VersaBot up and running in minutes

1

Clone Repository

git clone https://github.com/Scemworks/VersaBot.git
2

Navigate to Directory

cd VersaBot
3

Install Dependencies

pip install -r requirements.txt
4

Configure Bot Token

TOKEN="DISCORD_BOT_TOKEN"
5

Launch Bot

python main.py
